# 关于博客

 

建立博客是为了记录所学技术及工作经验，将知识传播，给网络上有所需要的朋友；

博客分类主要有建站技巧，资源分享，以及各类有趣的文摘，希望对您有所帮助。

博客宗旨：分享新意，收获快乐!

 

# 关于博主

 

博主学了相关计算机技术，目前做着平面设计以及网站建设相关工作。懂些电脑故障维修方面的知识，如果你有这方面的小问题，可以给我留言，或许我可以帮你解决系统相关故障，结交朋友。有新意吧博客更倾向于分享各类有新意的资讯及技巧，拒绝胡乱采集，长篇大论，绝不会拘泥于形式，图片、视频、音乐等，只要有好的内容，博主都会拿来丰富博客。。。

# 关于版权

 

有新意吧博客的博文均基于创作共享的知识共享署名-非商业性使用-相同方式共享 4.0 国际许可协议发布。 所有转载的博文将会在副标题内容中加注【转载:原文链接】超链接标识。而标注为【本站原创】的则为自己撰写或搜集整理并通过测试的教程或技术类文章，转载时请务必以超链接形式标明文章出处和作者信息。

若发现任何侵犯您利益的内容，请及时邮件或留言联系，我会第一时间删除所有相关内容。
